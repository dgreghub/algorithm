** 400만의 크기에 적합한 자료 구조 및 탐색 법
N이 2,000 인 경우 완전 탐색  N*N = 2,000 * 2,000 = 400만
N이 200,000인 경우, 부분배열 K크기인 N-K+1개의 특정값 i를  이진 탐색으로 찾는 경우 (N-K+1) * log2N  = 200,000 * 18 = 360만 
위와 같이, 탐색할 수 있는 자료의 범위가 곧 시간 복잡도를 뜻합니다.
